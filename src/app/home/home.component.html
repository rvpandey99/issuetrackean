This is a demonstration of an Issue Tracker web application created with Angular and Node(Express) as backend. You can click Issues button to view the previously raised issues. You will also see Update button before every entry, by clicking you will be able to update the issue or change the status.If you are using this app for first time, there may be no previously stored data. So, Please click Add New button to add some sample Issues.

  • To add new issues click on Add New Button.

  • To view issues click on Issues Button.
